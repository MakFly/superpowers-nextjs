# superpowers-nextjs skills map (lite)

Lightweight index for fast discovery.

## Core
- `nextjs:bootstrap-check` — Verify Next.js project configuration including next.config, TypeScript, ESLint, and App Router setup
- `nextjs:using-nextjs-superpowers` — Entry point for Next.js Superpowers - lightweight workflow guidance and command map.
- `nextjs:project-structure` — Organize Next.js projects with feature-based or layer-based folder structures
- `nextjs:runner-selection` — Select and configure the appropriate package manager (npm/yarn/pnpm/bun) based on project configuration

## Routing
- `nextjs:file-based-routing` — Understand file-based routing conventions with page.tsx, layout.tsx, loading.tsx, error.tsx, and not-found.tsx
- `nextjs:app-router` — Master Next.js App Router fundamentals - layouts, pages, templates, and the component hierarchy
- `nextjs:dynamic-routes` — Implement dynamic routes with [param], [...catchAll], and [[...optionalCatchAll]] patterns
- `nextjs:route-groups` — Organize routes with (group) folders for shared layouts and logical organization
- `nextjs:intercepting-routes` — Implement route interception with (.), (..), (..)(..), and (...) for modals
- `nextjs:route-handlers` — Create route handlers with Request/Response APIs, streaming, and edge runtime support for advanced API features

## Data & Caching
- `nextjs:mutations` — Implement optimistic updates, error handling, and rollback patterns for data mutations
- `nextjs:revalidation` — Master revalidatePath, revalidateTag, and on-demand revalidation for cache invalidation
- `nextjs:caching-strategies` — Master Next.js 16 caching with Cache Components, 'use cache' directive, and revalidation strategies
- `nextjs:session-management` — Manage sessions with cookies, server-side validation, and protected route patterns for secure user state
- `nextjs:data-fetching-patterns` — Implement data fetching with fetch(), cache(), and async components with proper error handling

## Server Components
- `nextjs:server-components` — Build with React Server Components (RSC) - async components, data fetching, and zero-JS patterns for optimal performance
- `nextjs:composition-patterns` — Apply component composition, render props, and compound components for reusable UI
- `nextjs:server-actions` — Create Server Actions with 'use server' for mutations, form submissions, and data modifications
- `nextjs:client-components` — Use 'use client' directive for interactive components with hooks, event handlers, and browser APIs

## Performance
- `nextjs:lazy-loading` — Implement lazy loading with dynamic imports, React.lazy, and Suspense boundaries
- `nextjs:streaming` — Use streaming SSR with loading.tsx, Suspense, and progressive rendering patterns
- `nextjs:font-optimization` — Load fonts optimally with next/font for self-hosted or Google Fonts with zero layout shift
- `nextjs:image-optimization` — Optimize images with next/image - responsive images, lazy loading, and format optimization

## Styling
- `nextjs:styled-components` — Integrate styled-components or CSS-in-JS solutions with App Router and RSC
- `nextjs:tailwind-integration` — Configure and use Tailwind CSS with Next.js including dark mode and custom themes
- `nextjs:css-modules` — Use CSS Modules for scoped, component-level styling with TypeScript support

## Testing
- `nextjs:e2e-playwright` — Write end-to-end tests with Playwright for full browser testing and visual regression
- `nextjs:testing-with-jest` — Configure and write tests with Jest and React Testing Library for Next.js applications
- `nextjs:testing-with-vitest` — Use Vitest for fast unit and integration testing with native ESM and TypeScript support
- `nextjs:react-testing-library` — Test React components with RTL best practices - queries, user events, and async testing

## Quality
- `nextjs:quality-checks` — Quality assurance checks for Next.js - ESLint, TypeScript, testing, performance, and code standards validation
- `nextjs:error-handling` — Implement error boundaries with error.tsx, global-error.tsx, and graceful degradation

## Other
- `nextjs:writing-plans` — Implementation planning for Next.js features - task breakdown, dependency analysis, and execution roadmaps
- `nextjs:authentication-patterns` — Implement authentication with NextAuth.js v5, JWT tokens, and server-side session validation for secure user management
- `nextjs:nextdevtools-mcp-integration` — NextDevTools MCP integration for Next.js 16 - development tools, server management, and debugging capabilities
- `nextjs:brainstorming` — Structured ideation and feature discovery for Next.js applications - design patterns, architecture decisions, and implementation strategies
- `nextjs:executing-plans` — Test-driven development execution for Next.js - implementing features through tests, TDD workflows, and quality assurance
- `nextjs:form-handling` — Build forms with useFormState, useFormStatus, and progressive enhancement patterns
