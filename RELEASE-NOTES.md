# Release Notes

## v0.1.0 (Initial Release)

### Features

- **37 Skills** covering all aspects of Next.js 16 development
  - App Router patterns and conventions
  - Server Components and Client Components
  - Server Actions and form handling
  - Data fetching and caching strategies
  - Testing with Jest, Vitest, and Playwright
  - Performance optimization
  - Styling solutions

- **14 Interactive Commands** for guided workflows
  - Brainstorming and planning
  - TDD workflows
  - Quality checks

- **Auto-Detection** of Next.js project configuration
  - Version and router type
  - Package manager
  - TypeScript settings
  - NextDevTools MCP

- **NextDevTools MCP Integration** for enhanced debugging

### Supported Versions

- Next.js 14.x, 15.x, 16.x
- npm, yarn, pnpm, bun
